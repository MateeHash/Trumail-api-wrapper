# Trumail

Trumail is a Python library for verifying emails.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Trumail.

```bash
pip3 install Trumail
```

## Usage

```python
import Trumail

Trumail.lookup("example@gmail.com")
```
